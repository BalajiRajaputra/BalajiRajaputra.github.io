Medicaid Drug Utilization & Reimbursements - Program Year 2017

About the Visualization

This visualization was presented by Balaji Rajaputra as the final project for CS 498 Data Visualization. This course is part of Masters in Computer Science (Data Science) at University of Illinois - Urbana Champaign

This project visualization is prepared based on the Medicaid utilization of program year 2017 data set.
Medicaid provides health coverage to millions of Americans, including eligible low-income adults, children, pregnant women, elderly adults and people with disabilities. Medicaid is administered by states, according to federal requirements. The program is funded jointly by states and the federal government.

Drug utilization data are reported by states for covered outpatient drugs that are paid for by state Medicaid agencies since the start of the Medicaid Drug Rebate Program. The data includes state, drug name, National Drug Code, number of prescriptions and dollars reimbursed. Data descriptions and data sets are available at : Medicaid.gov. & The data source.

1. Scenes :

The visualization uses hybrid interactive slide structure. The Choropleth map has zoom in/out options and hover over to the circle to see more details via tooltip. On the bar charts, the drop down option is available at the top of the page. By selecting different values from the drop down menu , parameters are changed and annotations are updated to present the new view. When hover over on the bar, the tooltip will provide details of the view

2. Annotations :

The block of text provided under each Slide has information that give insight into to the data. Tooltips, legends and labels will help user to understand the visualization.

3. Parameters :

There are 5 main parameters that filters the data and change the scene according to the selection. Key paramter values are - Number of Drugs Prescribed, Total medicaid amount reimbrsed, Number of Prescriptions, Units Reimburssed and Reimbursement in each quarter. The provided drop down menu give the option for user to change the parameter values.

4. Triggers :

The drop down menu selection triggers the change to the visualization. When the value is changed, the chart will be filtered and immedicately updated to show view based on new parameters.

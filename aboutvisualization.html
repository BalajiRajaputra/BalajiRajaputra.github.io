About the Visualization


1. Scenes :

  • The visualization uses hybrid interactive slide structure. This visualization has 1 choropleth and 2 bar charts.
  • The Choropleth map provides an overview of the Medicaid dug utilization for program year.
  • The map has zoom in/out options and user can hover over to the state to see more details via tooltip. The legend will help user to understand an overview of program utilization.
  • On the bar charts, the drop down option is available at the top of the page, it allows to pass different paramters for interactive visualization
  • By selecting different values from the drop down menu, parameters are changed and annotations are updated to present the new view. Hover over on the bar to find details

2. Annotations :

  • The block of text provided under each chart has information that give insight into to the data.
  • Tooltips, legend and labels will also help user to understand the visualization.

3. Parameters :

  • There are 5 main parameters on bar charts that filters the data and change the scene according to the selection.
  • Key paramter values are - Number of Drugs Prescribed, Number of Prescriptions, Number of Units Reimbursed and Reimbursements by each quarter.
  • The drop down menu provide an option for user to pass different parameter values to reflect on the interactive visualization.

4. Triggers :

  • The drop down menu selection triggers the change to the visualization.
  • When the value is changed, the chart will be filtered and immedicately updated to show new view based on new parameters.

About the Visualization



1. Scenes :

The visualization uses hybrid interactive slide structure. This visualization has 1 choropleth and 2 bar charts. The Choropleth map has zoom in/out options and user can hover over to the state to see more details via tooltip. The legend will help user to understand an overview of program utilization. On the bar charts, the drop down option is available at the top of the page. By selecting different values from the drop down menu , parameters are changed and annotations are updated to present the new view. When hover over on the bar, the tooltip will provide details of the view.

2. Annotations :

The block of text provided under each Slide has information that give insight into to the data. Tooltips, legend and labels will help user to understand the scene.

3. Parameters :

There are 5 main parameters that filters the data and change the scene according to the selection. Key paramter values are - Number of Drugs Prescribed, Total Medicaid Amount Reimbrsed, Number of Prescriptions, Units Reimbursed and Reimbursements by each quarter. The provided drop down menu give the option for user to change the parameter values.

4. Triggers :

The drop down menu selection triggers the change to the visualization. When the value is changed, the chart will be filtered and immedicately updated to show view based on new parameters.
